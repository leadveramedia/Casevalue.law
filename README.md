# CaseValue.law

CaseValue.law helps injured people estimate the potential value of their case through an interactive intake experience.

## Prerequisites

- Node.js 18+
- npm 9+

## Getting Started

```bash
npm install
npm start
```

The development server runs on `http://localhost:3000` and supports hot reloading.

## Quality Checks

- `npm test` runs the Jest test suite in watch mode.
- `npm run build` creates an optimized production bundle in the `build/` directory.

## Deployment

Deploy the contents of `build/` to your hosting provider or CDN of choice. Remember to include the assets in `public/`, such as `favicon.ico`, `social-share.jpg`, and `manifest.json`.
